# Common AI Response Errors

## Frequent Issues

### Hallucinations
Presenting false information as fact.

### Missing Context
Providing an answer without important background information.

### Overconfidence
Using absolute language despite uncertainty.

### Logical Inconsistency
Contradicting earlier statements or making unsupported conclusions.

### Incomplete Responses
Answering only part of the user's question.

### Outdated Information
Relying on information that is no longer current.

### Ambiguity
Using vague language that may confuse readers.

## Reviewer Checklist

- Is the response factually correct?
- Does it fully answer the prompt?
- Is the reasoning logical?
- Are unsupported claims present?
- Would additional context improve the response?
