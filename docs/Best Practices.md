# AI Evaluation Best Practices

## Purpose

This document outlines the principles followed when evaluating AI-generated responses.

## Best Practices

- Read the complete response before scoring.
- Verify factual claims using reliable sources when appropriate.
- Separate factual accuracy from writing quality.
- Avoid personal bias during evaluation.
- Consider the intended audience of the response.
- Reward clear reasoning, not just correct conclusions.
- Flag unsupported claims and overconfident statements.
- Suggest specific improvements rather than only identifying flaws.

## Goal

Produce evaluations that are consistent, fair, and actionable.
