# Contributing

Thank you for your interest in this project.

## How to Contribute

- Report issues or suggest improvements.
- Follow the evaluation framework and scoring rubric.
- Keep documentation clear and concise.
- Use the evaluation template for all new case studies.
- Ensure examples are factually accurate and well structured.

## Pull Requests

Please ensure any new case study:

- Follows the repository template.
- Includes an objective evaluation.
- Contains constructive recommendations.
- Maintains consistent formatting.
